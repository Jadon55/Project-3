All Available Routes:

  • /api/v1.0/table-name

    returns all data from the specified table

  • /api/v1.0/table-name/column1,column2

    returns selected column or columns from the specified table

  • /api/v1.0/table-name/column/data-value

    returns all rows that have the data-value in the specified column from the specified table
